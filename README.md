# 億萬人生 Billionaire Sandbox

可直接在瀏覽器遊玩的富豪模擬小遊戲。

## 內容

- 可搜尋並扮演 Forbes 2026 年度榜單全部 3,428 位富豪
- 50+ 種真實品牌與具體型號，提供品類、類型、品牌與文字搜尋
- 每項商品可一次購買多個，相同型號會自動合併數量
- 全球指數、債券、地產、科技、創投、加密資產等投資市場
- 每年推進、隨機市場波動、被動收入、資產配置與動態排行榜
- 每位角色獨立存檔，資料保存在瀏覽器

## 啟動

在此資料夾執行：

```powershell
python -m http.server 8080
```

再開啟 `http://localhost:8080`。

## 資料

角色初始資料採用 Gapminder 整理的 Forbes Billionaires dataset，2026-03-01 快照，共 3,428 人，CC BY 4.0。初始流動現金比例、投資報酬、購物價格皆為遊戲模擬。
